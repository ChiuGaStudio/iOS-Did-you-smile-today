# DAILY 的笑 - Did You Smile Today (iOS)

一款幫你每天「記得微笑」的小工具。用相機偵測你的笑容，留下當天的一張微笑，配合日曆回顧與溫柔提醒，讓微笑變成習慣。

## 主要功能

- 即時微笑偵測：相機畫面以指標顯示嘴角角度/曲線。
- 手動拍攝與預覽：確認後才存入，避免誤存。
- 日曆回顧：一眼看出哪天有「打卡」微笑。
- 每日提醒：可開關、可調時間，溫柔不打擾。
- 本機儲存：照片與紀錄只存於裝置中，不上傳。

## 如何使用

- 開啟 App，對準相機練習微笑，觀察指標變化。
- 當指標穩定上揚時按下快門，於預覽選擇「再次拍攝」或「確定存入」。
- 到「日曆」查看今天是否完成微笑打卡。
- 在「提醒」設定每日通知時間，幫自己記得微笑。

## 為什麼做

- 用小而確實的日常儀式，為自己留下片刻好心情。
- 尊重隱私，不社群、不上傳、零壓力。

## 隱私

- 所有影像與紀錄皆儲存在 App 沙盒中，不會離開你的裝置。
